# Olist-E-commerce-Analytics-Pipeline-de-Dados-End-to-End
Este projeto foi desenvolvido para analisar o desempenho de vendas, logística e satisfação de clientes da Olist. O diferencial deste trabalho é a construção de um pipeline de dados completo, que vai desde o tratamento de dados brutos com Python até a visualização avançada no Power BI.
